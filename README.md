### Project Description
This project contains Playwright automated tests for the SwiftTranslator web application.
The tests cover:
- 1 UI Test (Real-time output generation)
- 24 Positive Functional Test Cases
- 10 Negative Functional Test Cases

### Prerequisites
- Node.js (v18 or later)
- npm

### Installation
1. Clone the repository:
   git clone [<YOUR_GITHUB_REPO_LINK>](https://github.com/purnalakshitha99/playwright-_test_automation.git)

2. Navigate to the project folder:
   cd playwright

3. Install dependencies:
   npm install

4. Install Playwright browsers:
   npx playwright install

### Run Tests
To run all tests:
   npx playwright test

To run tests with browser UI:
   npx playwright test --headed

### View Test Report
After running tests:
   npx playwright show-report
